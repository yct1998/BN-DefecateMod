# BN-DefecateMod

The excretion management interface is located in **Action Menu → Misc**.

The excretion system is calculated based on changes in **thirst** and **calories**.

Using the toilet requires **paper**.

If you defecate in your pants, you will need **water in your inventory** to clean your body.

Unfortunately, since the filthy status has been removed, lower-body clothing cannot become filthy when a character soils their pants.
